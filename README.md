# C++ Modules

42 School C++ modules covering object-oriented programming fundamentals.

## Modules

### CPP00 - First Steps with C++
| Exercise | Description |
|----------|-------------|
| ex00 - Megaphone | Converts command-line arguments to uppercase |
| ex01 - PhoneBook | Interactive contact management system (up to 8 contacts) |
| ex03 - Account | Banking account class with static members and deposit/withdrawal tracking |

### CPP01 - Memory Allocation and References
| Exercise | Description |
|----------|-------------|
| ex00 - BraiiiiinnnzzzZ | Heap vs stack allocation with Zombie objects |
| ex01 - Moar brainz! | Dynamic array allocation (zombie horde) |
| ex02 - HI THIS IS BRAIN | Differences between strings, pointers, and references |
| ex03 - Unnecessary Violence | References vs pointers through a weapon system (HumanA/HumanB) |
| ex04 - Sed is for losers | File string substitution without using sed |
| ex05 - Harl 2.0 | Complaint filter using switch statements and method dispatch |

### CPP02 - Operator Overloading
| Exercise | Description |
|----------|-------------|
| ex00 - My First Class | Fixed-point number class (Orthodox Canonical Form) |
| ex01 - Towards a more usable class | Int/float conversions and stream output operator |
| ex02 - Now we're talking | Full arithmetic, comparison, and increment/decrement operators |

### CPP03 - Inheritance
| Exercise | Description |
|----------|-------------|
| ex00 - ClapTrap | Base robot class with attack, damage, and repair |
| ex01 - ScavTrap | Single inheritance with method overriding and guardGate() |
| ex02 - FragTrap | Extended inheritance hierarchy with highFivesGuys() |

### CPP04 - Polymorphism and Abstract Classes
| Exercise | Description |
|----------|-------------|
| ex00 - Polymorphism | Virtual functions with Animal/Cat/Dog classes |
| ex01 - Brain | Composition and deep copy with Brain objects |
| ex02 - Abstract Class | Pure virtual functions preventing base class instantiation |
